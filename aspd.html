SCHLAHUB

Web Development:

  1. Explain the differences between cookies, local storage, and session storage.
  2. What is CORS (Cross-Origin Resource Sharing), and how does it work?
  3. Describe the difference between HTTP and HTTPS protocols.
  4. How does client-side routing differ from server-side routing?

Mobile Development:

  1. What are the differences between native, hybrid, and web applications?
  2. Explain the purpose of Interface Builder in iOS development.
  3. What are the key components of Android architecture?
  4. Describe the differences between AsyncTask and Thread in Android.

Data Science/Machine Learning:

  1. Explain the bias-variance tradeoff in machine learning.
  2. What is regularization, and why is it important in training machine learning models?
  3. Describe the differences between supervised and unsupervised learning algorithms.
  4. What is gradient descent, and how does it work in optimizing machine learning models?

Cloud Computing:

  1. What is the difference between IaaS, PaaS, and SaaS?
  2. Explain the concept of horizontal and vertical scaling in cloud computing.
  3. What are the advantages of containerization over virtualization?
  4. Describe the key components of AWS (Amazon Web Services) architecture.

Cybersecurity:

  1. What is a DDoS (Distributed Denial of Service) attack, and how can it be mitigated?
  2. Explain the concept of encryption and its role in cybersecurity.
  3. What are the different types of malware, and how do they work?
  4. Describe the steps involved in a penetration testing (pen testing) process.

DevOps:

  1. What is Continuous Integration (CI), and how does it differ from Continuous Deployment (CD)?

  4. Describe the differences between Docker and Kubernetes.

Blockchain:

  1. What is a blockchain, and how does it work?
  2. Explain the concept of consensus mechanisms in blockchain technology.
