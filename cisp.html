DOMAIN WISE QUESTIONS

1. Software Development

Languages:

1. Explain the key differences between Java and Python.

2. What are the new features introduced in C# 9.0?

3. How do you manage memory in C++?

Data Structures & Algorithms:

1. How would you implement a binary search tree?

2. Describe the quicksort algorithm and its time complexity.

3. Explain how dynamic programming works with an example.

Design Patterns:

1. What is the Observer pattern? Provide an example.

2. How does the Factory pattern improve code flexibility?

3. Describe the use of the Decorator pattern in design.

Testing:

1. What is the purpose of integration testing?

2. How do you use mocks and stubs in unit testing?

3. Explain the Test-Driven Development (TDD) methodology.

2. Data Science & Analytics

Machine Learning:

1. What is the difference between classification and regression?

2. Explain the concept of overfitting and how to prevent it.

3. Describe the working of a decision tree algorithm.

1. What is the central limit theorem and why is it important?

2. How do you interpret a p-value in hypothesis testing?

3. Explain the difference between correlation and causation.

1. How would you handle missing values in a dataset?

2. What are the key differences between exploratory and confirmatory data analysis?

3. Describe the process of data cleaning and its importance.

Tools:

1. What are the primary features of the NumPy library in Python?

2. How do you use Matplotlib to visualize data?

3. Explain the advantages of using Jupyter Notebooks for data analysis.

2. What are the benefits of using serverless architecture?

3. Describe the different types of storage options available in AWS.

Security:

1. What are the best practices for securing cloud applications?

2. How do you implement identity and access management in Azure?

3. Describe the concept of encryption at rest and in transit.

Containers:

1. What is Kubernetes and how does it manage containerized applications?

3. How do you handle persistent storage in Docker containers?
