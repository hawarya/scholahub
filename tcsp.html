SCHLAHUB

TCS Domain-Specific Interview Questions

1. Software Development

Technical Questions

  • Explain the key differences between Java and Python.
  • Write a program to reverse a linked list.
  • How would you implement a binary search algorithm?
  • Can you explain the time complexity of sorting algorithms?
  • What are the four principles of OOP?
  • Give an example of how inheritance is used in your projects.
  • How would you normalize a database?
  • Write an SQL query to find the second highest salary from a table.

Scenario-Based Questions

  • Describe a challenging bug you encountered and how you resolved it.
  • How do you handle version control in a team project?

2. Data Science and Analytics

Technical Questions

  • Explain the difference between a population and a sample.
  • What is the p-value in hypothesis testing?
  • How does a decision tree algorithm work?
  • Compare and contrast supervised and unsupervised learning.
  • How do you handle missing values in a dataset?
  • Demonstrate how to merge two data frames in Python using pandas.
  • What is the best way to visualize categorical data?

Scenario-Based Questions

  • Explain a data science project you worked on from start to finish.
  • How do you choose the right model for a given dataset?

3. Cloud Computing

Technical Questions

  • Explain the differences between IaaS, PaaS, and SaaS.
  • How does AWS differ from Azure?
  • How would you deploy a web application on AWS?
  • Describe the use of Docker in cloud environments.
  • What are the best practices for securing data in the cloud?
  • How do you implement identity and access management in a cloud environment?

Scenario-Based Questions

  • Describe a time when you migrated an on-premises application to the cloud.
  • How do you handle scalability issues in cloud applications?

4. Networking

Technical Questions

  • Explain the difference between TCP and UDP.
  • What is the role of DNS in networking?
  • How do you set up a VLAN?
  • Describe the steps to configure a Cisco router.
  • What are the common network security protocols?
  • How do you prevent DDoS attacks?

Scenario-Based Questions

  • Describe a situation where you had to troubleshoot a network issue.
  • How do you ensure network reliability in a high-traffic environment?

5. Cybersecurity

Technical Questions

  • What is a zero-day exploit?
  • Explain the difference between a virus and a worm.
  • How do you implement multi-factor authentication?
  • Describe the steps to perform a security audit.
  • What is the difference between symmetric and asymmetric encryption?
  • How does SSL/TLS work?

Scenario-Based Questions

  • Describe a time when you dealt with a security breach.
  • How do you stay updated with the latest cybersecurity threats?

6. Business Analysis

Technical Questions

  • How do you gather requirements from stakeholders?
  • What techniques do you use to prioritize requirements?
  • Explain the use of UML in business analysis.
  • How do you create a business process model?
  • What tools do you use for project management and business analysis?

Scenario-Based Questions

  • Describe a project where your analysis significantly impacted the outcome.
  • How do you handle conflicts between stakeholders?

7. Quality Assurance (QA)

Technical Questions

  • Explain the difference between functional and non-functional testing.
  • What is regression testing?
  • How do you decide which tests to automate?
  • Describe your experience with any automation tools (e.g., Selenium, QTP).
  • What is your process for logging and tracking defects?
  • How do you prioritize bug fixes?

Scenario-Based Questions

  • Describe a challenging testing scenario and how you handled it.
  • How do you ensure test coverage and quality?
