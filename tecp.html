SCHLAHUB

Tech Mahindra Interview Questions

1. Software Development

Programming Languages (C, C++, Java, Python)

  • Write a C program to reverse a string without using any library functions.
  • How do you implement polymorphism in C++? Provide an example.
  • Describe how garbage collection works in Java.
  • Write a Python script to count the number of vowels in a given string.

Data Structures and Algorithms

  • What is the difference between merge sort and quick sort? Discuss their time complexities.
  • Write a program in C++ to find the middle element of a linked list in a single pass.
  • How would you implement a stack using two queues?
  • Describe a situation where you would use a hash table over a binary search tree.

Database Management

  • Write a SQL query to retrieve the names of employees who have the highest salary in each department.
  • Explain the different types of JOIN operations in SQL.
  • What are ACID properties in a database transaction?
  • How do you optimize a slow SQL query? Provide some techniques.

Web Technologies

  • Explain the difference between HTML4 and HTML5.
  • How do you implement a responsive design using CSS?
  • What is AJAX and how does it work in web development?
  • Explain the concept of closures in JavaScript.

2. Networking

Basics

  • Explain the OSI Model and the functions of each layer.
  • What are the main differences between IPv4 and IPv6?
  • How does a three-way handshake work in TCP/IP?
  • Describe the concept of subnet masking.

Protocols

  • Explain how HTTPS secures data transmission.
  • What is the function of the ARP protocol?
  • How does the SMTP protocol work for email transmission?
  • What are the differences between POP3 and IMAP?

Network Devices

  • Describe the main functions of a router and how it differs from a switch.
  • What is a firewall and what are its types?
  • How does a network bridge work?
  • Explain the concept of VLAN and its advantages.

3. Operating Systems

Processes and Threads

  • What are the differences between a process and a thread?
  • How do you handle deadlocks in an operating system?
  • Explain different types of scheduling algorithms.
  • What is context switching in operating systems?

Memory Management

  • Describe the concept of paging and how it is used in memory management.
  • What is the difference between internal and external fragmentation?
  • Explain the concept of segmentation in memory management.
  • How does a page replacement algorithm work? Discuss any one algorithm in detail.

File Systems

  • Explain how file permissions work in Unix/Linux.
  • What are the differences between NTFS and FAT32 file systems?
  • How does journaling work in a file system?
  • Describe the structure of an inode in Unix file systems.

4. HR Interview

Self-Introduction

  • Tell me about yourself.
  • What motivated you to apply for a position at Tech Mahindra?

Situational and Behavioral Questions

  • Describe a challenging project you worked on and how you managed it.
  • Give an example of a time you had to learn something quickly for a project.

Career Goals

  • What are your short-term and long-term career goals?
  • How do you see your career progressing in the next three years at Tech Mahindra?

Company Knowledge

  • What do you know about Tech Mahindra’s core values and culture?
  • How do you think Tech Mahindra’s services impact the industry?
