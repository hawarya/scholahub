DOMAIN WISE QUESTIONS

1. Software Development

Languages:

1. Explain the concept of OOP (Object-Oriented Programming) in C++.

2. What are the major differences between Python 2 and Python 3?

3. How do you manage memory in C?

Data Structures & Algorithms:

1. How do you implement a binary search tree?

2. What is the difference between Merge Sort and Quick Sort?

3. Explain how to detect a cycle in a linked list.

Design Patterns:

1. What is the Observer pattern and where is it used?

2. Explain the Model-View-Controller (MVC) architecture.

3. Describe the Decorator pattern with an example.

Testing:

1. What is Test-Driven Development (TDD) and its benefits?

2. How do you write test cases in Selenium?

3. Explain the concept of mocking in unit tests.

2. Data Science & Analytics

Machine Learning:

1. What are the steps involved in training a machine learning model?

2. Explain the concept of overfitting and how to prevent it.

3. How does the k-means clustering algorithm work?

Statistics:

1. What is the Central Limit Theorem?

2. Explain the difference between Type I and Type II errors.

3. How do you interpret a box plot?

Data Analysis:

1. Describe the steps you take to clean and preprocess data.

2. What is Principal Component Analysis (PCA)?

3. How do you handle missing values in a dataset?

Tools:

1. What are the main features of the NumPy library in Python?

2. How do you use Tableau for data visualization?

3. Explain the functionality of the Scikit-learn library.

3. Cloud Computing

Cloud Services:

1. What are the differences between IaaS, PaaS, and SaaS?

2. How do you implement serverless computing?

3. Explain the concept of cloud elasticity.

DevOps:

1. What is continuous integration and continuous deployment (CI/CD)?

2. How do you use Docker in a DevOps pipeline?

3. Explain the benefits of using Ansible for automation.

Security:

1. How do you secure an API?

2. What are the common vulnerabilities in cloud computing?

3. Describe the concept of identity and access management (IAM).

Containers:

1. What is the difference between Docker and Kubernetes?

2. How do you manage container orchestration?

3. Explain the concept of microservices architecture.

4. Networking

Protocols:

1. What is the difference between HTTP and HTTPS?

2. Explain how DNS works.

3. Describe the process of a TCP handshake.

OSI Model:

1. What are the functions of the Data Link Layer?

2. How does the Network Layer handle routing?

3. Compare the OSI model with the TCP/IP model.

Network Security:

1. What is a VLAN and why is it used?

2. Explain the concept of network segmentation.

3. How do you secure a wireless network?

Tools:

1. How do you use Nmap for network scanning?

2. Explain the features of NetFlow for network monitoring.

3. What is the use of traceroute in network diagnostics?

5. Cybersecurity

Threats:

1. What is SQL injection and how can it be prevented?

2. Explain the concept of a zero-day exploit.

3. Describe the different types of social engineering attacks.

Encryption:

1. What is end-to-end encryption?

2. How does a public key infrastructure (PKI) work?

3. Explain the concept of hashing and its uses.

Security Protocols:

1. How does the SSL/TLS protocol work?

2. What is IPsec and how is it used?

3. Describe the Kerberos authentication protocol.

Compliance:

1. What are the key requirements of the GDPR?

2. Explain the importance of SOC 2 compliance.

3. How does HIPAA influence data security in healthcare?

6. Database Management

SQL:

1. How do you write a SQL query to find duplicate records in a table?

2. What is the difference between DELETE and TRUNCATE in SQL?

3. Explain the concept of database transactions and ACID properties.

NoSQL:

1. What are the advantages of using a document-oriented database like MongoDB?

2. How does a column-family store like Cassandra work?

3. Explain the concept of eventual consistency in NoSQL databases.

Normalization:

1. Describe the process of normalizing a database to the second normal form (2NF).

2. What are the benefits of database normalization?

3. Explain the concept of transitive dependency in database design.

Performance Tuning:

1. How do you use indexing to improve database performance?

2. What are the best practices for query optimization?

3. Describe the role of database sharding in performance improvement.
